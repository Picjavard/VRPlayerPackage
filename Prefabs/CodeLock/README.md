# Кодовый замок

Кодовый замок из 5 переключаетелей (вкл/выкл). Стандартный код `^V^VV` (01011). События(`Events`) при открытии кодового замка нужно указать в скрипте `Boolean Event Proxy Emitter` объекта `Кодовый замок`
