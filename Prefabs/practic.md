**Обновленная инструкция: Создание локации для сборки монеток из unitypackage**  
**Цель:** За 60–80 минут собрать логический уровень с маршрутом для сбора монеток, используя готовые префабы.  

---

### **1. Подготовка проекта**  
1. **Создайте новую сцену:**  
   - *File → New Scene → 3D* → Сохраните как `CoinLevel` в папку `Scenes`.  
2. **Импортируйте префабы:**  
   - Скачайте unitypackage от учителя (файл `https://github.com/Picjavard/VRPlayerPackage/blob/main/Prefabs/Prafabs%20for%20blockout.unitypackage`).  
   - Испортируйте файл в Unity.  
   - В открывшемся окне нажмите **Import** (все галочки должны быть активны).  

---

### **2. Сборка уровня (25 минут)**  
**Задача:** Создать логический маршрут для сборки монеток.  
**Шаги:**  
1. **Основная платформа:**  
   - Перетащите **Floor2x2** в центр сцены (основа уровня).  
   - Добавьте **Floor1x1** сверху, чтобы создать подиум (высота = 2 блока).  
   - Используйте **Stairs1x2**, чтобы соединить уровни (пример: от пола к подиуму).  

2. **Препятствия и маршруты:**  
   - **Уклоны:** Добавьте **Slope1x2** и **Slope1x1** для создания наклонных поверхностей.  
   - **Стены:** Расставьте **Wall1x1** и **Wall2x2** так, чтобы игрок не мог «сойти с маршрута».  
   - **Пример маршрута:**  
     - Начало: `Floor2x2` → Стены по бокам → Наклон (`Slope1x2`) → Подиум с лестницей (`Stairs1x2`) → Финиш.  

3. **Размещение монеток:**  
   - Перетащите **Монетка** вдоль маршрута (минимум 5 штук).  
   - **Советы:**  
     - Положите одну монетку на подиуме — игрок должен подняться по лестнице.  
     - Разместите монетку на наклонной поверхности — проверьте, можно ли её собрать.  

---

### **3. Геймдизайн: Проверка проходимости (15 минут)**  
**Задание:** Убедитесь, что уровень логичен и проходим.  
1. **Проверьте маршрут:**  
   - Запустите сцену (*Play* → нажмите **F** на камере, чтобы центрировать на уровень).  
   - Пройдите маршрут мышкой (без скриптов) — все монетки должны быть доступны.  
2. **Исправьте ошибки:**  
   - Если монетка «в воздухе» — добавьте **Floor1x1** под неё.  
   - Если стены мешают — уменьшите их высоту (выделите стену → *Scale → Y*).  
3. **Добавьте сложность (по желанию):**  
   - Поставьте **Wall2x2** так, чтобы игрок обходил препятствие.  
   - Сделайте «ловушку»: разместите **Slope1x1** под монеткой — игрок должен прыгнуть.  

---

### **4. Механика сбора монеток (опционально, 15 минут)**  
**Если успеете — добавьте скрипт:**  
1. Создайте скрипт `CollectCoins.cs` в папке `Scripts`:  
   ```csharp  
   using UnityEngine;  
   public class CollectCoins : MonoBehaviour {  
       public int collected = 0;  
       void OnTriggerEnter(Collider other) {  
           if (other.CompareTag("Player")) {  
               collected++;  
               Destroy(gameObject);  
               Debug.Log("Монеток собрано: " + collected);  
           }  
       }  
   }  
   ```  
2. **Настройте монетку:**  
   - Выделите префаб **Монетка** в папке `Prefabs`.  
   - В *Inspector*:  
     - Добавьте компонент **Box Collider** (Is Trigger = **true**).  
     - Прикрепите скрипт `CollectCoins`.  
     - Установите тег **Coin** (*Tag → Edit Tags → + → Coin*).  
3. **Проверьте:**  
   - У игрока (куб-«герой») должен быть тег **Player** и компонент **Rigidbody**.  

### **Итог урока:**  
1. Сохраните сцену.  
2. **Скриншоты:** Сфотографируйте уровень и схему маршрута (на бумаге или в Paint).  
3. **Вопросы для обсуждения (5 минут):**  
   - «Какие препятствия вы добавили? Почему они улучшают игру?»  
   - «Что сложнее: разместить монетки или сделать маршрут проходимым?»
